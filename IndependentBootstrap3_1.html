Twitter Bootstrap
Свободный набор инструментов для создания сайтов и веб-приложений
Включает в себя HTML и CSS шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейсов, включая JavaScript расширения.
Сетки — заранее заданные размеры колонок, которые можно сразу же использовать, например, ширина колонки 140px относится к классу span2, который можно использоваться в JavaScript описании документа.
Типогрфика — описание шрифтов, определение некоторых классов для шрифтов, таких как код, цитаты и т.п.,
Медиа — представляет некоторое упаравление изображениями и видео.
